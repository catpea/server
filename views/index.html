Temporary News Bites

The Printer Friendly pages for poetry work nicely now, there is a cover page and a link references page that will always print separately so that the texts of my long-winded poems are stand-alone. But it makes no sense to print out video pages, it can be done as an art project, perhaps printing out film-strips of videos, but printing video-pages won't be so useful, in reality.

The real poetry print layout is only available for viewing in print-preview/print-to-file. Even though it is a page meant for printing, it still has to be styled for the browser, as it looks ugly when styled for the printer and shown on the screen. So, even though it looks all white and text friendly, it is still made for the browser, and the pretty print layout is seen in print preview and on paper. It wasn't hard to make, but it is not as simple as stripping-off colors and graphics.

As to links, I still wanted to display them when printing on paper, somehow. So I borrowed the Wikipedia way, I marked all the links with numbers, and then made a little lookup table on the bottom. Printing the web addresses next to the links was distracting and messed with layout. You can take a look at the link rich I Miss Old Programming Languages for a worst possible scenario (note the link-mess on the bottom). And check out By Subtle Analogy and Big Things for something more reasonable.
